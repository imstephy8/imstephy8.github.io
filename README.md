# imstephy8.github.io

My personal portfolio website built to introduce who I am, showcase my projects, and share my academic and career interests as a Computer Science and Applied & Computational Mathematics student.

This website serves as my personal online space where visitors can learn more about me, my background, my projects, and my journey toward becoming a software engineer.

The design goal of this website is to keep it simple, clean, and easy to navigate, while still reflecting my personal style.

- Personal introduction
